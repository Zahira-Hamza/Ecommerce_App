## ✨ Features

- 🔐 **Secure Authentication** — JWT-based login and registration with session persistence
- 🛒 **Shopping Cart** — Add, remove, and manage cart items with real-time total calculation
- 📦 **Order Management** — Track orders in real-time with live status updates
- 🔍 **Product Search & Filtering** — Search by keyword, filter by category, price range, and more
- 🗂️ **Category Browsing** — Navigate products organized by category for easy discovery
- 💳 **Checkout Flow** — Smooth multi-step checkout with order confirmation
- 📱 **Responsive UI** — Pixel-perfect design that adapts to all screen sizes

## 🚀 Getting Started

### Prerequisites
- Flutter SDK `>=3.0.0`
- Dart SDK `>=3.0.0`

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Zahira-Hamza/Ecommerce_App.git

# 2. Navigate into the project
cd Ecommerce_App

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run
```

## 📁 Project Structure

```
lib/
├── core/               # Constants, themes, utilities
├── models/             # Data models (Product, Order, User, Cart)
├── services/           # API service layer
├── providers/          # State management (Provider)
├── screens/            # UI pages
│   ├── auth/           # Login, Register
│   ├── home/           # Home, Categories
│   ├── product/        # Product detail, Search
│   ├── cart/           # Cart, Checkout
│   └── orders/         # Order history, tracking
└── widgets/            # Reusable UI components
```
